# Pagina-web-do-zero-primeira-interacao

**Tema:** Atrizes tailandesas (GL Thai)

**Objetivo do projeto:**
Desenvolver um site moderno e responsivo dedicado às atrizes tailandesas do gênero GL (Girls' Love). O objetivo é reunir informações organizadas sobre as atrizes, suas carreiras, filmografias, séries em que atuaram, curiosidades, prêmios, galerias de imagens e outras informações relevantes para fãs e interessados na cultura GL tailandesa.

**Público-alvo:**
Fãs de séries GL tailandesas, pessoas interessadas em conhecer novas atrizes e usuários que desejam acessar informações de forma rápida e organizada.

**O que o projeto oferece:**

* Perfis completos das principais atrizes GL Thai.
* Biografia e trajetória profissional.
* Filmografia e séries em que participaram.
* Curiosidades e fatos interessantes.
* Galeria de fotos.
* Campo de pesquisa para encontrar atrizes rapidamente.
* Interface moderna, intuitiva e responsiva.
* Navegação simples e agradável para dispositivos móveis e computadores.

**Tecnologias utilizadas:**

* HTML para a estrutura da aplicação.
* CSS para estilização, responsividade e animações.
* JavaScript para interatividade, pesquisa dinâmica, filtros e manipulação do conteúdo.

**Requisitos de design:**

* Visual elegante inspirado na cultura tailandesa.
* Paleta de cores suaves (rosa, lilás, branco e dourado).
* Cartões (cards) para exibir cada atriz.
* Animações suaves ao passar o mouse.
* Layout totalmente responsivo.
* Ícones modernos e tipografia limpa.

**Funcionalidades desejadas:**

* Página inicial com destaque para as atrizes mais populares.
* Barra de pesquisa em tempo real.
* Sistema de filtros por série ou atriz.
* Página individual com informações detalhadas de cada atriz.
* Seção de curiosidades.
* Rodapé com informações do projeto.

**Resultado esperado:**
Criar um site informativo, visualmente atrativo e fácil de navegar, oferecendo uma experiência agradável para fãs de atrizes GL tailandesas e servindo como uma referência para conhecer melhor suas carreiras e trabalhos.
